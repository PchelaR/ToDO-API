Эндпоинты ToDo API

Это API приложение предназначено для управления задачами. Оно позволяет пользователям создавать, читать, обновлять и удалять задачи, а также управлять аутентификацией и регистрацией пользователей.

Как управлять задачами

Изменение задачи

Для изменения задачи используйте эндпоинт /api/v1/todo/{id}/ с методом PUT. Передайте обновленные данные задачи в теле запроса.

Удаление задачи

Для удаления задачи используйте эндпоинт /api/v1/todo/{id}/ с методом DELETE. Задача будет удалена по указанному ID.
